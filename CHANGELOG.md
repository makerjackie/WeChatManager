# 更新记录

## 1.1.4 - 2026-07-18

- 移除与微信自身能力重复的文件管理和缓存清理，主界面更专注于多开与分身管理。
- 不再访问微信聊天文件目录，并移除“访问其他 App 数据”权限及相关引导。
- 首次设置简化为微信应用和应用管理两步，普通多开、方案与兼容增强能力保持不变。
- 精简 README 与宣传图，减少无关信息。

## 1.1.3 - 2026-07-18

- 兼容增强会明确显示当前微信是否可用，并在详情中列出全部已适配版本。
- 当前版本不支持时，推荐微信 4.1.5.28，并优先提供腾讯官方精确构建下载，历史版本作为备用。
- 增加推荐版本 SHA256 校验信息，避免下载到版本号相近但内部构建不匹配的安装包。
- 兼容判断会同时检查处理器架构，避免在 Intel Mac 上误报 Apple 芯片补丁可用。
- 主视觉恢复为绿色，继续使用独立的多窗口图形，不使用微信聊天气泡标志。

## 1.1.2 - 2026-07-18

- 主界面改为结论优先，移除大段常驻说明文字。
- 为分身、文件、更新、方案、隐私和兼容增强增加按需查看的信息按钮。
- 权限引导简化为三步“下一步”流程，系统授权成功后自动进入下一步。
- 权限失败时只显示简短结论，具体原因收进信息按钮。
- 文件列表默认只显示名称、操作和占用空间，路径与目录说明按需查看。
- 修正分身发现测试读取真实“应用程序”目录的问题，测试环境现在完全隔离。

## 1.1.1 - 2026-07-18

- 更换独立的多窗口应用图标与蓝绿色视觉，不再接近微信原图标。
- 精简首页、分身、文件、方案、权限与设置界面的说明文案。
- 支持从已被其他本机工具修改或重新签名的微信创建和更新分身。
- 创建前仍会校验微信 Bundle ID 与可执行文件，避免误将其他应用作为微信复制。
- 兼容增强继续保留腾讯签名、精确版本和完整备份的安全限制。

## 1.1.0 - 2026-07-18

- 新增“方案与更新”中心，自动识别与官方微信版本不一致的分身，并支持一键同步全部分身。
- 批量更新继续使用固定 Bundle ID 与独立账号容器，旧分身应用移入废纸篓，登录状态与聊天数据不删除。
- 支持选择常用分身保存为方案，并在本机或新 Mac 上创建缺少的分身、更新版本或恢复显示名称。
- 新增 iCloud 键值同步，只同步方案元数据；聊天、登录状态、文件路径和账号标识始终不同步。
- iCloud 不可用时自动使用本地副本，恢复方案不会删除方案外的分身。
- 新增方案冲突选择、本地回退、精确序号恢复与显示名称保留测试。

## 1.0.4 - 2026-07-18

- 开发版改用独立 Bundle ID，避免本地测试签名覆盖正式版的 macOS 隐私授权记录。
- 微信文件目录改为由应用进程直接读取，不再通过限时子进程反复触发“访问其他 App 数据”请求。

## 1.0.3 - 2026-07-18

- 增加首次启动中文权限引导：先解释用途，再分别请求微信应用信息与微信文件访问权限。
- 增加“应用管理”分步说明，明确系统会在首次实际创建、更新、移除分身或执行兼容增强时请求允许。
- 说明管理员授权、分身摄像头、麦克风和通知权限的触发时机与使用边界。
- 在设置页增加重新查看权限引导及直接打开“应用管理”设置的入口。
- 优化系统权限用途文案，并补充权限状态与引导持久化测试。

## 1.0.2 - 2026-07-18

- 将新建微信分身安装到系统“应用程序”文件夹，避免微信提示从安装包直接打开；旧分身会在更新或下次启动时迁移。
- 文件管理改为按微信账号分组，默认使用“微信号一、微信号二”等友好名称，并支持自定义命名。
- 打开位置和复制路径改为直接按钮；完整数据路径默认收起。
- 精简文件列表信息，账号标识不再直接出现在常规界面。
- 补充 macOS 应用数据权限说明，并避免系统权限确认期间阻塞界面刷新。
- 增加账号别名隐私、分身位置迁移及文件展示测试。

## 1.0.1 - 2026-07-18

- 改善微信启动逻辑对不同 Xcode 和 Swift 6 工具链的兼容性。
- 更新 GitHub Actions 运行时，保持持续集成环境处于受支持版本。

## 1.0.0 - 2026-07-18

- 首个公开版本。
- 提供原生 SwiftUI 中文界面、独立账号容器的微信分身与多实例管理。
- 支持创建、启动、更新和安全移除分身，微信更新时保留分身登录数据。
- 自动定位微信 4.x 与旧版文件、视频、账号和缓存目录。
- 支持磁盘占用计算与移入废纸篓的安全缓存清理。
- 提供按构建号严格匹配、完整备份、签名校验和恢复的兼容增强。
- 集成 Sparkle 2 自动检查、下载、安装和重启更新。
- 提供 Universal 2、Developer ID 签名、Apple 公证 DMG。
