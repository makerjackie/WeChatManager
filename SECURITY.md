# 安全说明

## 报告安全问题

请不要公开披露仍可利用的漏洞。通过 GitHub 仓库的 Security Advisory 私下报告，并附上复现条件、受影响版本和预期影响。

## 安全边界

- 应用不读取聊天内容，不上传微信数据。
- 缓存清理使用精确路径白名单，并移动到废纸篓。
- 兼容增强只接受 HTTPS 上游配置、精确构建号和明确目标。
- 修改微信前保留腾讯官方签名的完整备份。
- 自动更新需要 Sparkle EdDSA 签名、Developer ID 签名与 Apple 公证。

兼容增强会改变微信代码签名，无法消除微信服务条款或账号策略带来的风险。不了解该影响时，请只使用普通多开和文件管理。
